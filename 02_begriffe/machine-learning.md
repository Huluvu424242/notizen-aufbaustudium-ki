# Machine Learning

## Kurzdefinition

Machine Learning beschreibt Verfahren, bei denen Modelle aus Daten Muster lernen, statt dass alle Regeln explizit programmiert werden.

## Zentrale Aspekte

- Trainingsdaten
- Modell
- Lernverfahren
- Validierung
- Generalisierung

## Praxisbezug

Für Softwarearchitektur ist wichtig zu verstehen, dass ML-Komponenten nicht deterministisch wie klassische Programmlogik funktionieren.
