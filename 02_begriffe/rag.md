# Retrieval-Augmented Generation

## Kurzdefinition

Retrieval-Augmented Generation kombiniert ein generatives Sprachmodell mit einer vorgeschalteten Suche in externen Wissensquellen.

## Zweck

Das Modell soll Antworten nicht nur aus seinem Trainingswissen erzeugen, sondern relevante Dokumente oder Datenquellen einbeziehen.

## Typische Bausteine

- Dokumentensammlung
- Chunking
- Embeddings
- Vektordatenbank
- Retriever
- Prompt
- LLM

## Praxisfragen

- Wie aktuell sind die Quellen?
- Wie erklärt man die Herkunft einer Antwort?
- Wie verhindert man Halluzinationen?
