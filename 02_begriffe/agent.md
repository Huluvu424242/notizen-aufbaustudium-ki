# Agent

## Kurzdefinition

Ein Agent ist ein System, das Ziele verfolgt, Informationen verarbeitet, Entscheidungen trifft und Aktionen ausführen kann.

## Im KI-Kontext

Bei LLM-Agenten wird ein Sprachmodell häufig mit Werkzeugen, Speicher, Planung und Ausführungsschritten kombiniert.
