# Embedding

## Kurzdefinition

Ein Embedding ist eine numerische Repräsentation von Daten, z.B. Text, Bildern oder Begriffen, in einem Vektorraum.

## Warum wichtig?

Embeddings sind zentral für:

- semantische Suche
- RAG
- Ähnlichkeitsvergleiche
- Wissenssysteme
- LLM-Anwendungen
