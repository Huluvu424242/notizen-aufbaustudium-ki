# Künstliche Intelligenz

## Kurzdefinition

KI ist ein Forschungsgebiet mit dem Ziel, Systeme zu entwickeln, 
die lernen, logisch denken, und unabhängig handeln, 
wenn sie mit neuen Situationen konfrontiert sind.

## Beziehung zu Machine Learning

Machine Learning (ML) ist ein Teilgebiet der KI.

ML beschreibt Verfahren, bei denen Systeme aus Daten Muster lernen,
anstatt dass sämtliche Regeln explizit programmiert werden.

## Persönliche Einordnung

Der aktuelle KI-Boom scheint stark durch Fortschritte im Machine Learning
und insbesondere im Deep Learning ausgelöst worden zu sein.

Dadurch wurde KI:
- sichtbarer,
- wirtschaftlich relevanter,
- und im Alltag breiter nutzbar.

Viele moderne KI-Systeme basieren heute auf ML-Verfahren.
