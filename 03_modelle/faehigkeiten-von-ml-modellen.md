# Fähigkeiten von ML-Systemen

## Kontext

Im Kurs wurde beschrieben, dass Machine-Learning-Systeme
verschiedene Arten von Fähigkeiten erlernen können.

Die genaue Klassifikation scheint didaktisch motiviert zu sein.

## Persönliche Interpretation

Die dargestellten Fähigkeiten erinnern mich stark an eine
technische Nachbildung menschlicher Wahrnehmungs- und
Ausdrucksfähigkeiten.

Dabei scheint es Überschneidungen mit den menschlichen
7 Sinnen zu geben.

Mögliche Analogie:

| Menschliche Fähigkeit | KI-/ML-Fähigkeit                                  |
|-----------------------|---------------------------------------------------|
| Hören                 | Computer Audition (Audio- und Sprachverarbeitung) |
| Riechen               |                                                   |
| Schmecken             |                                                   |
| Sehen                 | Computer Vision (Bildverarbeitung)                |
| Tastsinn              |                                                   |
| ---                   | ---                                               |
|                       | Computer Linguistik                               |
|                       | Robotik & Steuerung                               |
|                       | Vorhersage                                        |
|                       | Entdeckung                                        |
|                       | Planung                                           |
|                       | Generierung                                       |

## Beobachtung

Moderne generative KI erweitert klassische Wahrnehmungssysteme
um die Fähigkeit, neue Inhalte zu erzeugen.

Dadurch entsteht der Eindruck, dass KI-Systeme nicht mehr nur
"verstehen", sondern zunehmend auch "erschaffen" können.

## Offene Fragen

- Ist Generierung wirklich eine eigene Fähigkeit oder nur eine Folge guter  
  Modellierung?
- Gibt es eine etablierte wissenschaftliche Klassifikation solcher Fähigkeiten?
- Welche Fähigkeiten benötigen zwingend Deep Learning?
- Welche Fähigkeiten existierten bereits vor modernen LLMs?
