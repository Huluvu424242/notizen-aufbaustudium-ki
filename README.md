# KI-Studium Notizen

Persönliches Markdown-Repo zur Begleitung eines modularen Einstiegs in KI, z.B. über KI-Campus-Kurse, Zertifikate oder ein späteres Aufbaustudium.

## Ziele

- KI-Grundlagen strukturiert aufbauen
- Begriffe, Modelle, Methoden und offene Fragen sammeln
- Lernfortschritt nachvollziehbar dokumentieren
- praktische Experimente sauber ablegen
- spätere Anrechnungsmöglichkeiten und Studienoptionen vorbereiten

## Empfohlene Arbeitsweise

1. Pro Kurs einen eigenen Ordner unter `01_kurse/` verwenden.
2. Neue Begriffe zusätzlich unter `02_begriffe/` ablegen.
3. Offene Fragen sofort erfassen, auch wenn sie noch unscharf sind.
4. Praktische Experimente getrennt von Kursnotizen dokumentieren.
5. Wöchentlich kurz reflektieren: Was habe ich verstanden? Was ist offen? Was ist relevant für meine Arbeit?
