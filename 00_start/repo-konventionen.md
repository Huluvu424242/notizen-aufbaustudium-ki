# Repo-Konventionen

## Dateinamen

- Kleinbuchstaben
- Wörter mit Bindestrich trennen
- Keine Leerzeichen
- Beispiel: `neuronale-netze.md`

## Notizformat

Jede inhaltliche Notiz sollte enthalten:

- Datum
- Quelle/Kurs
- Zusammenfassung
- zentrale Begriffe
- offene Fragen
- persönliche Einordnung
- Bezug zur Praxis

## Tags

Mögliche Tags:

- `#grundlagen`
- `#machine-learning`
- `#deep-learning`
- `#llm`
- `#agenten`
- `#rag`
- `#wissenssysteme`
- `#mathe`
- `#praxis`
- `#offene-frage`
